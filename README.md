# Python Quiz Game

A simple command-line quiz game built with Python. The program asks users general knowledge questions, checks their answers, keeps track of the score, and displays the final result.

## Features

- Interactive command-line interface
- Multiple quiz questions
- Score tracking
- Instant feedback after each answer
- Final score displayed at the end

## Technologies Used

- Python 3

## How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/PriyalAmeta/python-quiz-game.git
   ```

2. Open the project folder:

   ```bash
   cd python-quiz-game
   ```

3. Run the program:

   ```bash
   python3 python_quiz.py
   ```
